\# Reviewer Skill



\## Purpose



Evaluate outputs produced by all agents.



Generate quality scores and recommendations.



\---



\## Trigger



Run after Summary Agent, Task Agent and Email Agent complete.



\---



\## Responsibility



Evaluate only.



Never modify outputs.



Never generate new content.



\---



\## Inputs



Summary



Tasks



Email

\---



\## Evaluation Criteria



Evaluate



\- Completeness

\- Accuracy

\- Consistency

\- Missing information

\- Hallucinations



\---



\## Constraints



Never change outputs.



Only evaluate.



Scores between 0 and 100.



\---



\## Quality Checklist



✓ Summary complete



✓ Tasks complete



✓ Email references summary



✓ Email references tasks



✓ No hallucinations
